# Tic-Tac-Toe

This is a simple Tic-Tac-Toe game designed using HTML, CSS, and JavaScrip

## How to Play

Tic-Tac-Toe is a two-player game. The players take turns marking the cells on a 3x3 grid. One player uses "X" and the other uses "O" to mark their moves. The player who successfully places three of their marks in a horizontal, vertical, or diagonal row wins the game.

## Features

- Simple and intuitive user interface.
- Keeps track of the current player's turn.
- Displays a message when the game ends, showing the winner or a draw.

## Installation

No installation is required to play the game online. However, if you want to run the game locally on your machine, follow these steps:

1. Clone the repository to your local machine using `git clone https://github.com/your-username/tic-tac-toe.git`.

2. Change into the project directory by running `cd tic-tac-toe`.

3. Open the `index.html` file in your web browser.
